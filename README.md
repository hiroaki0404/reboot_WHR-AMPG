# reboot_WHR-APMG

reboot_WHR-APMG は、Buffaloの無線ルータ [WHR-AMPG](http://buffalo.jp/products/catalog/network/whr-ampg/)を再起動させる、[JMeter](http://jmeter.apache.org/)のスクリプトです。

## 動作確認バージョン
WHR-AMPG Ver.1.52 (R1.03/B1.0.37-1.19-1.05)
